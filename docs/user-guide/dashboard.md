# Dashboard Overview

The EAS Station dashboard provides real-time monitoring of emergency alerts and system status.

## Dashboard Components

### Alert Map
Interactive Leaflet map showing active alerts with geographic coverage areas.

**Features:**
- Color-coded alert markers by severity
- Click markers for alert details
- Zoom and pan controls
- Multiple map layers

### Statistics Panel
Real-time statistics including:
- Active alerts count
- Alerts in last 24 hours
- Total alerts processed
- System uptime

### Recent Alerts List
Chronological list of recent alerts with:
- Event type and severity
- Affected areas
- Issue and expiration times
- Quick action buttons

### System Status
Health indicators for:
- Database connection
- Alert poller status
- Last poll time
- Error notifications

## Navigation

Main navigation provides access to:
- **Admin** - System administration
- **Settings** - Configuration
- **System Health** - Detailed diagnostics
- **Help** - Documentation

See [Managing Alerts](alerts.md) for alert operations.
