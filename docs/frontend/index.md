# Frontend Documentation

User interface, components, and frontend development for EAS Station.

## Overview

EAS Station features a modern, responsive web interface built with:

- **Framework**: Bootstrap 5
- **Maps**: Leaflet with OpenStreetMap
- **Charts**: Highcharts
- **Icons**: Font Awesome 6

## Key Features

### Responsive Design
Mobile-first design that works on all devices

### Dark Mode
Built-in light/dark theme support

### Real-time Updates
Live alert updates without page refresh

### Interactive Maps
Leaflet-powered geographic visualization

### Accessibility
WCAG 2.1 AA compliant interface
